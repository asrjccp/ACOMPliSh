# ACOMPliSh

---
## About
ACOMPliSh is a prompt-engineered generative AI web app to help A-level students learn H2 Computing. We are always work in progress. :) 

---
## Structure
```
ACOMPliSh/
├── pages (name)/
│   ├── chapter1.py
│   ├── chapter2.py
│   └── ...
├── images/
│   └── chapter1/
│       ├── picture1.png
│       ├── picture2.png
│       └── ...
├── code (for RAG)/
│   └── chapter1/
│       ├── code1.py
│       ├── code2.py
│       └── ...
├── .streamlit/
│   └── secrets.toml
└── requirements.txt
```
